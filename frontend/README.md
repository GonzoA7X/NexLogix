# NexLogix - Logistics 

Plataforma Full-Stack diseñada para la monitorización, trazabilidad y gestión logística empresarial. Implementa una arquitectura moderna basada en Monorepo para asegurar escalabilidad y separación de responsabilidades.

## Arquitectura del Sistema

- **Backend:** Python 3.10+, FastAPI (Endpoints RESTful de alto rendimiento), Pydantic (Validación de esquemas y tipado de datos). Configuración estricta de políticas CORS.
- **Frontend:** React 18 (Vite JS), Zustand (Gestión de estado global asíncrono), Recharts (Visualización de datos SVGs dinámicos), UI Responsiva estilo "Dark Theme" mediante CSS-in-JS nativo.

## Características Principales
- **Dashboard Interactivo:** Visualización de KPIs en tiempo real (Volumen total, operaciones en ruta, incidencias).
- **Gestión de Estado Asíncrono:** Manejo eficiente de estados de red (loading/error/success) mediante Fetch API nativo acoplado a Zustand.
- **Arquitectura Escalable:** Estructura de componentes modulares y lista para futura integración con bases de datos SQL.
